# TAPe UTility

Command line tool for processing TAP files.

## Usage
<pre>
Usage: TAPUT command [options] FileIn [FileOut]<br>
commands:
       add
	   insert
       extract
       remove
       list
       help
options:
       -o &lt;address&gt;
       -n &lt;block name&gt;
       -b &lt;block number&gt;
</pre>
